Essential Glossary

Repo (Repository): The project folder containing your code and history.

Remote: The version of your project on the internet (GitHub/GitLab).

HEAD: A pointer showing which commit/branch you are currently looking at.

Conflict: When two people change the same line of code and Git needs you to choose a winner.
